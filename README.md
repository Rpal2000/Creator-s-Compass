# Creator-s-Compass
A multimodal AI agent built with Gemini &amp; Opus that solves creators' core video problems. Users upload a link, and the system instantly analyzes lighting, audio, and framing to generate a Problem: Solution report. It features an auditable workflow, human review checks, and Qdrant-powered style recommendations.
